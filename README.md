# Temperature-Converter
This project implements a temperature conversion system in C++ that allows users to convert values between Celsius and Fahrenheit through an interactive console-based menu.

## Features
- Celsius to Fahrenheit conversion
- Fahrenheit to Celsius conversion
- Continuous execution using loop
- Input-based menu system

## Concepts Used
- Loops (while)
- Conditional statements (if-else)
- Constants
- User input/output
